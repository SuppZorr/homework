Кнопка (тег button) с текстом "OK" по центру.
Размер кнопки 120пикс*45пикс.
Кнопка имеет округления по углам в 5 пикселей, цвет кнопки - градиент. 
При нажатии кнопка меняет свой цвет на другой цвет, тоже градиент.  
Значения цветов  сгенерировано на сайте  http://colormind.io/
Кнопка отбрасывает тень, когда не нажата.
