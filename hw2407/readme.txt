Страница, сверху блок заголовка, снизу блок основной. 
В основном блоке   текст. 
В заголовке слева   изображение-логотип, справа 3 блока выпадающего меню. 
Меню сделано таким образом:
1) при изменении ширины браузера остается с правой стороны, 
2) при изменении ширины текста в любом пункте меню, оно автоматом подстраивалось 
3) при наведении мышки на выпадающий любой пункт меню слева рисуется зонтик.
